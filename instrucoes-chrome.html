Instruções para usar o Burlesco no Chrome

O Google removeu o Burlesco da Chrome Web Store por considerá-lo uma violação das suas diretrizes. Desde então, o Chrome não é oficialmente suportado pelo projeto.

Como muitos usuários insistem em usar o Chrome, há um método alternativo de instalação. O problema é que você reberá avisos pedindo para desativar o Modo do desenvolvedor sempre que iniciar o navegador. Além disso você não terá atualizações automáticas.

Seguem as instruções:

  1. Baixe o arquivo burlesco-chrome.zip
  2. Descompacte
  3. Digite chrome://extensions/ na barra de endereços do Chrome
  4. Ative o Modo do desenvolvedor no canto superior direito
  5. Clique em Carregar extensão expandida...
  6. Selecione a pasta onde o burlesco-chrome foi descompactado

Não tem como remover o aviso?

O Google quer ter controle sobre o seu navegador e decidir por você o que deve ou não ser instalado nele. A melhor maneira de resolver esse problema é mudar para um navegador que respeita sua liberdade.

Se você só quiser remover o aviso chato, instale o Chrome Canary (versão experimental). Uma opção menos radical, porém mais trabalhosa, é editar o Group Policy do Windows. O processo é intencionalmente trabalhoso para desencorajar usuários a exercerem sua liberdade.
