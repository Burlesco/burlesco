Usar a extensão do Burlesco no Chrome

Você pode usar a extensão no Chrome instalando manualmente. O problema é que você reberá avisos pedindo para desativar o Modo do desenvolvedor sempre que iniciar o navegador. Além disso você não receberá atualizações automáticas.

Seguem as instruções:

  1. Baixe o arquivo burlesco-chrome.zip
  2. Descompacte
  3. Digite chrome://extensions/ na barra de endereços do Chrome
  4. Ative o Modo do desenvolvedor no canto superior direito
  5. Clique em Carregar extensão expandida...
  6. Selecione a pasta onde o burlesco-chrome foi descompactado

Não tem como remover o aviso?

O Google quer ter controle sobre o seu navegador e decidir por você o que deve ou não ser instalado nele. A melhor maneira de resolver esse problema é mudar para um navegador que respeita sua liberdade.

Outra opção é instalar o Chrome Canary (versão experimental). Se preferir usar a versão estável, é possível remover o aviso editando o Group Policy do Windows.
