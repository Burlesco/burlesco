Instruções para usar o Burlesco no Edge

A Microsoft ainda não aprovou a extensão do Burlesco para o Microsoft Edge.

Felizmente você pode instalar o nosso userscript! A experiência do usuário ainda é um pouco inferior a extensão nativa, mas todos os sites suportados pela extensão também funcionam com o userscript.

Seguem as instruções:

  1. Instale a extensão Tampermonkey
  2. Depois de instalar o Tampermonkey, instale o userscript
  3. Pronto! Agora é só aproveitar o Burlesco!

E se eu quiser usar a extensão?

A melhor maneira de resolver esse problema é mudar para um navegador que respeita sua liberdade.
